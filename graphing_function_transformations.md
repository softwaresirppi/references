# Multiply A with X
	Scales along -X axis A units
	Mirrors along the axis if A is negative
# Add A with X
	Move along -X axis A units.
# Rotate
	90 deg => (x, y) -> (y, -x)
	-90 deg => (x, y) -> (-y, x)
	180 deg => (x, y) -> (-x, -y)
	270 deg => (x, y) -> (-y, x)
	-270 deg => (x, y) -> (y, -x)

NOTE: +ve clockwise -ve anticlockwise
